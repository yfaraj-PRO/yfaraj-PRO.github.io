# Portfolio - Yassine

Site single-page léger pour Yassine (2ème année ingénierie en informatique, Fès, Morocco).  
Le site est écrit en HTML/CSS/JS pur et est prêt pour GitHub Pages.

## Fichiers
- `index.html` — page principale
- `style.css` — styles
- `script.js` — interactions (menu mobile, smooth scroll, formulaire)
- `assets/` — (optionnel) met tes images/screenshots ici
- `README.md` — ce fichier

## Personnalisation (À FAIRE)
1. Remplace `youremail@example.com` dans `index.html` et `script.js` par ton email réel.
2. Remplace les liens `href="#"` dans la section socials par tes URL GitHub, LinkedIn et Fiverr.
3. Remplace les images placeholder par de vraies captures dans `assets/`.

## Prévisualiser localement
Ouvre `index.html` dans ton navigateur (double‑clic) — aucun serveur nécessaire.

## Déployer sur GitHub Pages
1. Merge la branche `feature/portfolio-site` dans `main`.
2. Sur GitHub : Settings → Pages → Source → sélectionne `main` (root) → Save.
3. Ton site sera disponible à : `https://yfaraj-PRO.github.io` (attendre 1–2 minutes).
